# Outrank Personal

Личный ИИ-инструмент для автоматизации создания, публикации и анализа контента (SEO-статьи, Reels, Shorts, TikTok-видео) на русском, украинском и английском языках.

## Основные возможности
- Исследование ключевых слов и трендов
- Генерация SEO-статей, изображений и коротких видео
- Автоматическая публикация в WordPress, X/Twitter, Facebook, Instagram Reels, TikTok, YouTube Shorts, Reddit
- Реал-тайм тренды и AI-идеи постов
- Аналитика постов, PDF-отчёты, мониторинг комментариев + автоответы
- Расписание публикаций, бренд-кит, безопасность ключей
